# 🧮 SmartCalc — The Student Calculator

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Active-brightgreen?style=for-the-badge)

A powerful all-in-one calculator built for students and everyday use — goes far beyond basic math. Built with pure Python, no libraries needed.

---

## ✨ Features

| Module | What it does |
|---|---|
| 📐 Basic Calculator | Full expression evaluator with brackets & powers |
| 📏 Unit Converter | Length, weight, temperature, speed, area, data |
| 🔢 Quadratic Solver | Solves ax²+bx+c=0 including complex roots |
| 📊 Statistics | Mean, median, mode, std deviation, variance |
| 💯 Percentage Tools | Tax, discount, percentage change, and more |
| 🔣 Number Tools | Prime check, factorial, Fibonacci, GCD/LCM, base converter |
| 📜 History | Tracks your last 20 calculations |

---

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/ETCREALMAN/smart-calculator.git
cd smart-calculator

# Run (zero dependencies!)
python smart_calculator.py
```

## 💡 Why SmartCalc?

Most calculators just do arithmetic. SmartCalc solves the real problems students face daily — from unit conversions to quadratic equations to statistics homework — all in one place, with no internet required.
